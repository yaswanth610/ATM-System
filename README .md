# 🏧 ATM Management System

## 📌 Overview
This is a simple ATM Management System built using Python. It is a menu-driven console application that simulates basic ATM operations.

## ✨ Features
- 🔐 ATM Login
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Balance
- 🔑 Change Security PIN
- ❌ Invalid Input Handling

## 🛠 Technologies Used
- Python 3

## ▶️ How to Run

1. Download or clone this repository.
2. Open a terminal.
3. Run:

```bash
python atm.py
```
